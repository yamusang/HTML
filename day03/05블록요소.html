블록요소 - div,p,pre, h1~h6

블록요소는 문서의 내용을 표시하는 구역을 설정합니다. 시맨틱 태그(header,nav,main,article...) div와 동일한 성격을 갖고 의미있는 단어를 사용한 것 뿐입니다.

블록요소는 기본적으로 width(너비)는 브라우저 크기의 100%, height(높이)는 자식요소의 크기입니다. 블록요소의 크기를 변경하려면 width, height 스타일을 직접 설정합니다.

문서 최상위 요소인 body는 브라우저 너비 100%, 높이 자식요소의 크기

div 첫번째 테스트 - px 단위를 사용하면 항상 고정된 크기를 설정합니다. (절대값)
div 두 번째 테스트 - % 단위를 사용하면 부모요소의 크기를 기준으로 설정합니다.(상대값) 현재 div의 부모 요소는 body. body의 width(너비) 는 브라우저의 100%입니다. body의 높이(height)는 자식요소의 크기만큼으로 설정됩니다.
